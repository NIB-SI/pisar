
# R Package *pisar*

R support for pISA-tree

[![DOI](https://zenodo.org/badge/214719940.svg)](https://zenodo.org/badge/latestdoi/214719940)

## Installation


```R
if (!require(devtools)) {
    install.packages("devtools")
    require(devtools)
}
devtools::install_github("NIB-SI/pisar", build_vignettes = TRUE)
```
