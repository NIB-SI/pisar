
# R Package *pisar*

R support for pISA-tree

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.5721249.svg)](https://doi.org/10.5281/zenodo.5721249)

## Installation


```R
if (!require(devtools)) {
    install.packages("devtools")
    require(devtools)
}
devtools::install_github("NIB-SI/pisar", build_vignettes = TRUE)
```
